# Surveilans PKM Cibuntu Bandung

Sistem surveilans (pengawasan) terintegrasi untuk PKM Cibuntu Bandung dengan fitur monitoring real-time, manajemen pengguna, dan pelaporan.

## Fitur Utama

- ✅ Dashboard interaktif dengan statistik real-time
- ✅ Live monitoring dari multiple kamera
- ✅ Sistem alert otomatis
- ✅ Manajemen pengguna dengan role-based access
- ✅ Laporan terperinci dengan export PDF
- ✅ Sistem notifikasi (Email, SMS, Push)
- ✅ Activity logging untuk audit trail

## Tech Stack

### Frontend
- HTML5, CSS3, JavaScript (ES6+)
- Chart.js untuk visualisasi data
- Responsive design (Mobile-friendly)

### Backend
- Node.js + Express.js
- MySQL untuk database
- JWT untuk authentication
- bcryptjs untuk password hashing

### Database
- MySQL 8.0+
- Optimized queries dengan indexes
